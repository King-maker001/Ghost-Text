# 👻 ghost text

A high-performance, lightweight browser extension that dynamically expands custom keyword shortcuts into full code blocks or text strings across textareas, inputs, and web IDEs.

## 🚀 How to Install on Your Laptop

Since this extension is shared directly via code, you can load it into Chrome in less than 60 seconds:

1. Click the green **Code** button at the top right of this page and select **Download ZIP**.
2. Extract/Unzip the downloaded `ghost-text-main.zip` folder onto your Desktop.
3. Open Google Chrome and type **`chrome://extensions/`** into your URL bar.
4. Look in the top-right corner of that page and turn the **Developer mode** toggle switch **ON**.
5. Click the **Load unpacked** button in the top-left corner.
6. Select the extracted `ghost-text` folder (the folder that contains the `manifest.json` file).

## 💡 How to Use
* Type default shortcuts like `;sysout` or `;psvm` in any text box or web coding editor to watch them instantly expand.
* Click the **Ghost Text** icon in your Chrome extensions toolbar to open the manager panel, where you can add your own custom shortcuts and delete old ones!
